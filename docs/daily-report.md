## 二、深度观察专栏

### QDII溢价趋势与跨境套利风险提示

**事件概述**：近期美股在AI算力热潮驱动下持续创新高，纳斯达克100指数与标普500指数不断刷新历史高位。强烈的跨境配置需求与QDII额度受限的矛盾，导致场内跨境ETF溢价率急剧攀升。截至最新数据，纳指ETF国泰溢价率高达**9.45%**，纳指ETF广发溢价**8.75%**，纳斯达克100ETF招商溢价**7.25%**；标普500ETF国泰与博时溢价率也分别达到**5.14%**与**4.89%**。与此同时，场外QDII申购额度依然紧俏，建信纳斯达克100多只份额日累计限额已降至**100元**，汇添富相关产品限额为**50元**。

**深层分析**：
1. **供需失衡与额度瓶颈**：外汇管理局对QDII额度发放保持克制，而国内投资者对美股科技资产的需求呈刚性增长。场外申购大门几乎关闭，迫使资金涌入场内ETF，导致二级市场交易价格大幅偏离基金净值。这种高溢价本质上是“稀缺性”的变现，而非基金本身资产的真实增值。
2. **套利机制失效与踩踏风险**：理论上，ETF高溢价会吸引套利资金通过申购份额在二级市场抛售套利，从而抹平溢价。但由于QDII额度受限，外汇申购通道受阻，一级市场套利机制失效，溢价得以长期维持。然而，一旦美股出现阶段性回调或QDII额度意外放宽，场内溢价将迅速收敛，高溢价接盘的投资者将面临“净值下跌+溢价坍塌”的双杀风险。

**影响与前瞻**：当前纳指与标普500场内ETF的异常高溢价，已透支未来数月的收益预期。对于持有高溢价ETF的投资者，建议警惕溢价收敛风险，可考虑逐步切换至溢价较低或平价的场外指数基金（尽管限额较低，但无溢价损耗）。下周若美国CPI或PCE数据超预期引发美股科技股回调，高溢价品种将首当其冲，建议回避场内溢价超**5%**的跨境ETF，防范流动性踩踏。
