A股收盘（7月15日）

A股主要指数多数回调，上证指数收报3955.58点（↓0.29%），深证成指报14779.40点（↓0.97%），沪深300报4786.78点（↓0.20%），创业板指报3804.70点（↓1.21%），科创50重挫收报1924.27点（↓4.25%）。两市合计成交额约2.57万亿元，较前5日均值有所放大。全市场上涨家数2997家，下跌1647家，涨跌比为1.82，涨停72家，跌停31家。申万一级行业涨跌数据暂不可得，但从主力资金流向看，证券（净流入67.51亿）、银行（51.23亿）、化学制药（41.06亿）获资金青睐，而半导体（净流出124.63亿）、通信设备（92.23亿）、光学光电子（80.28亿）遭遇大幅抛售。全市场主力资金净流出378.54亿元。板块轮动呈现明显的“弃高就低”风格，资金从科技成长切向金融、医药与白酒等防御及消费板块。
