二、行业洞察

全球 TOP 10

  1. SK海力士创海外赴美最大IPO，首秀收涨13%：SK海力士在纳斯达克完成规模超越阿里的IPO，首日股价大涨约13%。CEO称2027年将是内存最短缺年份，缺货或持续至2030年，彰显AI算力对存储芯片的庞大需求。 → 利好全球AI硬件及存储产业链，强化算力景气周期预期。（Fortune）
  2. 美联储半年度报告警示关税与AI推高通胀：美联储发布政策报告，指出关税和中东地缘局势正推高通胀，同时AI既是增长动力，又因推升电力、内存等成本而成为通胀“双刃剑”。高盛预计AI涨价或推高核心PCE 0.5个百分点。 → 影响偏鹰，利空美债，延缓美联储降息节奏。（第一财经）
  3. 特朗普向美国企业施压以降低物价：白宫正向沃尔玛等顶级杂货商施压，要求其降低牛肉等商品价格。特朗普政府试图通过政治压力对抗通胀，企业利润率面临考验。 → 利空必需消费品板块，企业定价权与毛利率或受挤压。（Financial Times）
  4. 大众汽车大幅削减产品线以应对中国销量下滑：大众汽车宣布将裁减一半产品线并缩减产能，因在中国市场销量暴跌及电动车转型压力剧增。公司未明确提及裁员计划。 → 利空传统车企，反映中国市场竞争对海外巨头冲击加剧。（CNBC）
  5. Meta探索AI云业务，股价转涨：Meta宣布探索AI云业务并公布数据中心扩张计划及新AI模型定价策略，扎克伯格称进军AI云业务“顺理成章”，推动股价年内转涨。 → 利好AI算力与云服务板块，加剧科技巨头AI军备竞赛。（Yahoo Finance）
  6. 阿里合计持股长鑫科技近5%，成重要股东：阿里巴巴合计持股长鑫科技近5%，持股比例超过董事长朱一明。长鑫科技作为A股年内最大IPO，发行价定为4.41元，引发市场对产业资本与散户博弈的热议。 → 利好国产存储芯片产业，凸显互联网巨头加码硬科技投资趋势。（华尔街见闻）
  7. 阳光电源发布EnerNeo固态变压器，瞄准AI数据中心：阳光电源签定SST战略协议，管理层预计2028年进入大规模商用部署。公司针对AI算力中心高电耗痛点，推出800V直流架构固态变压器，争取2026年实现订单落地。 → 利好储能与算力供电设备赛道，打开光储企业新增长曲线。（第一财经）

  9. 商业航天迎利好，概念股集体涨停：受行业重大利好刺激，A股航天装备概念板块异动拉升，航天环宇20%涨停，航天电子触及涨停板，资金加速涌入商业航天产业链。 → 利好商业航天及卫星互联网板块，主题催化特征显著。（证券时报）
  10. 诺和诺德联手Vivani研发长效GLP-1植入物：诺和诺德与Vivani达成协议，评估长效GLP-1植入物，该植入物可维持长达一年的药效，旨在改善患者长期减重依从性。 → 利好创新药与减重药赛道，GLP-1给药方式革新提升竞争壁垒。（CNBC）
