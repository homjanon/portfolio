一、市场全景

A股收盘（7月10日，周四）

A股主要指数昨日全线大涨，以TMT与新能源为首的成长股引领反弹。

上证指数报收3996.16，上涨1.00%。深证成指报收15046.67，上涨2.29%。沪深300报收4780.79，上涨1.96%。科创50报收2064.98，上涨5.53%。创业板指报收3842.73，上涨4.37%。
