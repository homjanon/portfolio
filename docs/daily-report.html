美股收盘（7月15日凌晨）

道琼斯工业指数收报52498.64点，↓0.26%；标普500收报7515.34点，↓0.79%；纳斯达克综合收报25873.18点，↓1.55%。

宏观层面，美国6月CPI放缓至3.5%，汽油价格回落带来通胀降温信号。主要驱动因素呈现多空博弈：高盛和摩根大通等华尔街大行Q2财报创纪录盈利，AI交易狂潮推升股票交易收入；但IBM因AI资本开支转移预警暴跌25%，创史上最大单日跌幅，拖累软件及科技板块。标普500十一大板块中，金融板块受财报提振领涨，科技软件板块受IBM拖累承压。

港股收盘（7月14日）

恒生指数收报24340.73点，↑0.52%；恒生国企指数收报8103.08点，↑0.46%。南下资金大幅净买入109.97亿港元，重点加仓大模型AI龙头及互联网科技股。

全球其他市场（7月14日）

欧洲STOXX 600指数微跌↓0.01%，基本持平。日经225指数下跌↓1.92%。韩国KOSPI指数遭遇重挫，暴跌↓8.95%，高盛交易员报告指出韩国股市正经历"持续且痛苦"的去杠杆，外资与机构联手出逃，最高46万账户遭强平，存储芯片巨头大跌引发半导体板块巨震。
