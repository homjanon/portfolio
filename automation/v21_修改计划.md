**当前**：雪球蛋卷API 提供 PE/PB/分位/股息率（5个指数）

**优化**：
```python
def fetch_industry_v21():

### 11.2 逐板块优化详述

#### 📍 优化1：A股收盘 → 升级为 "A股收盘 + 全市场温度 + 风格诊断"

**当前问题**：
- 申万行业TOP3和主力资金流依赖 WebSearch，数据来源不稳定，格式不统一
- 缺少全市场估值温度指标
- "板块轮动判断"依赖 LLM 自由发挥，缺少数据锚点

**优化方案**：

| 子项 | v20 方式 | v21 方式 | 收益 |
|------|---------|---------|------|
| 申万行业TOP3/BOTTOM3 | WebSearch 搜索 | `data_industry.json` → "申万TOP3"/"申万BOTTOM3" 字段 | 数据精确量化，格式统一 |
| 主力资金流 | WebSearch 搜索 | `data_industry.json` → "资金_净流入TOP5"/"资金_净流出TOP5" | 同花顺90行业，净流入列 |
| 🆕 全市场PE温度 | 不存在 | `data_industry.json` → "全市场PE" | 日更PE温度计 |
| 风格诊断 | LLM 自由发挥 | 基于31行业涨跌幅自动判断：大盘vs小盘、价值vs成长 | 有数据锚 |

**输出示例**（v21）：
```
